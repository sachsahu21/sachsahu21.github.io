Things I've built

Multi-country sales data pipeline

Python · FTP · Hologres

Designed and maintained an automated pipeline ingesting daily sales files from multiple countries over FTP, each with its own schema and column conventions. Built country-aware parsing logic in Python/pandas and loaded standardized output into Hologres to power a regional sales dashboard.

Unified reporting across markets with a single daily refresh

Data normalization engine for reference codes

MaxCompute SQL · regex

Source systems delivered nationality and reference codes in wildly inconsistent formats — mixed leading zeros, trailing punctuation, extra digits. Built a robust chained REGEXP_REPLACE + SUBSTR normalization layer in ODPS SQL, iterated against real-data edge cases until output was clean and deterministic.

Eliminated manual cleanup and downstream join failures

GST implementation — Malaysia

Development · User support

Delivered both the development and user-support tracks of a national GST implementation project in Malaysia. Worked directly with business stakeholders, raising flags early on serious issues and cross-checking requirements before building — earning a strong reputation across the department.
